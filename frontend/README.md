# StageApp - Application de Gestion des Stages

## Description
Application React pour la gestion des stages médicaux avec trois types d'utilisateurs :
- **Administrateurs** : Gestion des étudiants et hôpitaux
- **Étudiants** : Gestion des demandes de stage
- **Hôpitaux** : Gestion des candidatures
